# WxTUI

## What is WxTUI?

WxTUI is a free, open source terminal user interface that uses National Weather Service (NWS) data to bring up live alerts/warnings. Displaying this information using [Textual](https://github.com/Textualize/textua). WxTUI allows users to customize what alerts they see.

## Who made it?

Simon Crystal is a weather nerd and freshman at Carrboro High School, he made this TUI so that he could see live warnings during **tornado season**. See his progress on [Hackclub Macondo](https://macondo.hackclub.com/projects/3801).

## How to use it?

Install textual & python, than download these files, or clone this repo and run ```python WxTUI/app.py```
